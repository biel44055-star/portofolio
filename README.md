# Nabiel Arinaullah Portfolio

Interactive dark-gradient portfolio website ready for deployment to **Vercel** or **GitHub Pages**.

## Contents
- `index.html` — main portfolio page
- `styles.css` — all styling
- `script.js` — animations, interactivity, gallery modal, particles
- `assets/` — profile and certificate images
- `docs/` — CV and supporting PDF documents
- `vercel.json` — simple Vercel config

## Highlights
- Dark gradient UI with animated particles
- Responsive layout
- Sections for:
  - About
  - Skills
  - Work Experience
  - Organization & Leadership
  - Awards & Achievements
  - Selected Projects
  - Portfolio Gallery
  - Supporting Documents
  - Contact
- Gallery built from the ZIP portfolio images
- Ready for direct upload to GitHub or Vercel

## Deploy to Vercel
1. Upload this folder to GitHub.
2. Import the repository into Vercel.
3. Deploy as a static site.

## Deploy to GitHub Pages
1. Push this folder to a repository.
2. Open repository **Settings → Pages**.
3. Choose deployment from the main branch root.
4. Save and publish.
